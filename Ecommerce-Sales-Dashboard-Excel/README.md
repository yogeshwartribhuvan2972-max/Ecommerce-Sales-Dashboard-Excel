# E-commerce Sales Dashboard using Excel

This project analyzes one year of e-commerce sales data using Microsoft Excel.

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key Insights
- Sports and Clothing are top revenue-generating categories
- Bangalore and Pune generate highest sales
- Digital payments dominate over cash transactions
- Faster delivery improves customer ratings

## Business Recommendations
- Focus marketing on top categories
- Improve low-performing regions
- Reduce delivery delays
- Increase digital payment offers
